# AWS Boto3 SDK Python Automation

This repository contains Python scripts I wrote while learning how to automate AWS infrastructure tasks using the Boto3 SDK.

The goal of this repository is to demonstrate basic cloud automation skills, including interacting with AWS services programmatically, managing cloud resources, and understanding how infrastructure can be controlled through code.

## Skills Practiced

- Python scripting
- AWS Boto3 SDK
- EC2 instance management
- S3 CRUD operations
- Boto3 clients and resources
- VPC, internet gateway, and routing concepts
- Basic infrastructure automation
- Cloud operations fundamentals

## Repository Contents

Examples in this repository include:

- Importing and configuring Boto3
- Demonstrating Boto3 clients and resources
- Creating, reading, updating, and deleting AWS resources
- Performing CRUD operations with S3
- Launching EC2 instances
- Starting, stopping, and terminating EC2 instances
- Managing EC2 resources
- Working with VPCs, internet gateways, and routing
- Creating a database cluster

## Purpose

I work in IT operations and am building my programming and cloud automation skills to better understand how infrastructure can be managed, monitored, and automated.

This repository represents hands-on practice with AWS automation concepts that are useful for cloud operations, system administration, cybersecurity operations, and DevOps-style workflows.

## Security Note

No real AWS credentials, access keys, secret keys, or sensitive configuration values are included in this repository.

Any scripts that interact with AWS should be reviewed and tested carefully in a safe lab environment before use.
